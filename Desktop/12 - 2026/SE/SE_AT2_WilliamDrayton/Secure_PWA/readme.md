- How to run the PWA -

1. -> Open the VScode terminal.

2. -> Create a virtual enviroment by entering "python3 -m venv venv".

3. -> Activate the virtual enviroment by entering "\venv\Scripts\Activate.ps1" on Windows or "source venv/bin/activate" on Mac/Linux.

4. -> Run flask by typing "flask run" into the venv.

5. -> Open your browser and type the url: "http://127.0.0.1:5000".

6. -> Create an account and sign in.
