- Welcome to Thyme Machine - Your PWA Recipe Book. -

- App Overview -

ThymeMachine is a Progressive Web App Recipe System that allows users to:

- Signup for an account and login at anytime
- Create, view, edit and save recipes
- Add custom ingredients
- Favourite recipes
- Filter by cuisine type and favourites

- Features -

Account System :

- Users signup with a username, email and password
- Log in to create and save recipes
- Data Saved to an SQLite Database

Recipe Management :

- Create Recipes with a name, cuisine type, cooking time, ingredients list, instructions and an optional notes section
- Create Custom Ingredients with a name, amount and a unit
- Recent recipes are displayed on the homepage

Filtering System :

- Users recipes have the option to be filtered by favourites and cuisine type

Accessibility :

- Flexible measurement units for ingredients
- Bold text option in My Profile Page
- Responsive Design for all devices

- Instructions For Use -

1. Sign up for an account using your email and a unique password and username
2. Log in to ThymeMachine
3. On the homepage locate the navbar and click on "Add New" to create a new recipe
4. Create and save your first recipe 
5. View your new recipe by clicking on the "My Recipes" link in the navbar
6. Cook and enjoy your new ThymeMachine Recipe
